# NattHUB 🚀
> A Premium, Modern Glassmorphism UI for Roblox.

![NattHUB Mockup](https://raw.githubusercontent.com/nztta/natthub/main/logo_mockup.png) <!-- Note: User would need to upload mockup -->

NattHUB is a high-performance, visually stunning script hub designed with a modern glassmorphism aesthetic. It features advanced teleportation, interactive dropdowns, and a responsive design.

## ✨ Features
- **Glassmorphism UI**: Beautiful semi-transparent frames with linear gradients.
- **Modern Palette**: Custom Off-Black and Off-White theme.
- **World Teleport**: Quick access to all game portals with a sleek dropdown interface.
- **Background Blur**: Immersive experience with dynamic lighting integration.
- **Draggable & Responsive**: Fully interactive and optimized for a clean look.

## 🚀 Installation
Paste the following code into your Roblox script executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/nztta/natthub/main/natt_hub.lua"))()
```

## 🛠️ Configuration
To use your own logo, edit the `Config` table in `natt_hub.lua`:

```lua
Config = {
    LogoID = "rbxassetid://YOUR_ID_HERE", -- Your Roblox Asset ID
    -- ... other settings
}
```
